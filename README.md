Calculator Web Application

Level 2 – Task 1
Web Development & Design Internship (OIBSIP)

This project is a basic Calculator Web Application developed using HTML, CSS, and JavaScript. The objective of this task is to build a functional calculator with a clean user interface and proper handling of user input and arithmetic operations.

Live Demo

https://jisha-christopher.github.io/OIBSIP-Level-2-Task-1/

Project Description

The calculator allows users to perform basic mathematical operations such as addition, subtraction, multiplication, and division.
The application includes a display screen to show user input and calculation results. CSS Grid is used for proper alignment of buttons, while JavaScript manages the calculation logic and user interactions.

Features

Basic arithmetic operations (+, −, ×, ÷)

Interactive display for input and output

Clean and user-friendly interface

Responsive design for different screen sizes

Technologies Used

HTML5

CSS3

JavaScript
